# Secret-Word

Secret-Word is an interactive game developed with React, where players try to guess a secret word based on hints. The game offers a fun and challenging experience, ideal for group entertainment.
## 🚀 Features

- 🎮 Interactive Game: Join a dynamic word-guessing game.
- 🤔 Hints: Receive hints to help guess the secret word.
- 🏆 Scoreboard: Keep track of scores and see who the best player is!
  
## 🛠️ Technologies Used

This project was built using the following technologies:

- React: For building the user interface and managing state.
- CSS: For styling and responsive design.
- HTML: For structuring the content.
  
## 📥 Installation

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
  ```bash
  git clone https://github.com/gustavobanares/Secret-Word
```
2. Navigate to the project directory:
```bash
cd secretword
```
3. Install the dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```
5. Open http://localhost:5173/ in your browser to play Secret Word.
